3. System Information

top : Display real-time system processes and resource usage

df : Show disk space usage
// df -h (human-readable format)

du : Show disk usage of files and directories
// du -sh directory_name

free : Show memory usage
// free -h

uname : Show system information
// uname -a (displays full system info)


4. File Permissions and Ownership

chmod : Change file permissions
// Note : Give read, write and execute permissions to owner; read and execute to others

chown : Change file owner and group
// chown user:group filename

ls -l : List files with detailed information including permissions and ownership
